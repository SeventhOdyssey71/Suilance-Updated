zkLogin

zkLogin is a Sui primitive that provides the ability for you to send transactions from a Sui address using an OAuth credential, without publicly linking the two.

Overview

zkLogin is designed with the following goals in mind:


  • Streamlined onboarding: zkLogin enables you to transact on Sui using the familiar OAuth login flow...

  • Self-custody: A zkLogin transaction requires user approval...

  • Security: zkLogin is a two-factor authentication scheme...

  • Privacy: Zero-knowledge proofs prevent third parties from linking...

  • Optional verified identity: A user can opt in to verify the OAuth identifier...

  • Accessibility: zkLogin integrates with other Sui primitives...

  • Rigorousness: The code for zkLogin has been independently audited...
